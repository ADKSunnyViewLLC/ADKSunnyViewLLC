Welcome to the official GitHub repository for **ADK Sunny View LLC**, a thoughtfully planned manufactured home community located in Peru, New York. This repository hosts the project website, including conceptual designs, site plans, development strategy, and community vision.

---

## Project Overview

ADK Sunny View LLC aims to develop **16 new manufactured home sites** and a **community center** at an existing residential property in the North Country. The project emphasizes:

- **Affordable, quality housing** suitable for families and individuals
- **Low-density planning** to preserve green spaces and privacy
- **Sustainable development** with long-term land stewardship
- **Community building** through shared spaces and a central community center
